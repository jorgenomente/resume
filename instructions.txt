genial — si ya tenés el proyecto y librerías, te dejo el paquete mínimo de archivos + estilo para que tu CV tome el look “oscuro + glass + acentos teal/cobre (Futurismo Sensible)” y quede listo.

0) Paleta y concepto visual

Fondo: petroleum dark con ligero ruido y gradientes sutiles.

Tarjetas: glass (blur + borde sutil + brillo suave).

Acentos: Teal #7DAA92 y Cobre #C27C53.

Radio grande (rounded-2xl), sombras suaves y micro-animaciones en hover.

5) Componentes UI (glass + badges + timeline)

components/Card.tsx

import { cn } from "@/src/lib/utils";

export default function Card({
  className,
  children,
}: React.PropsWithChildren<{ className?: string }>) {
  return (
    <div
      className={cn(
        "glass rounded-2xl p-6 shadow-soft hover:shadow-glow transition-shadow duration-300 ease-smooth",
        className
      )}
    >
      {children}
    </div>
  );
}


components/Badge.tsx

export default function Badge({ children }: { children: React.ReactNode }) {
  return (
    <span className="inline-flex items-center gap-1 rounded-full border border-line bg-glass/60 px-3 py-1 text-sm text-white/90">
      {children}
    </span>
  );
}


components/Section.tsx

export function SectionTitle({ children }: { children: React.ReactNode }) {
  return (
    <h2 className="text-xl font-semibold tracking-wide text-white/90">
      <span className="text-grad">{children}</span>
    </h2>
  );
}

export function SectionWrap({ children }: { children: React.ReactNode }) {
  return <div className="space-y-4">{children}</div>;
}


components/TimelineItem.tsx

export default function TimelineItem({
  title,
  subtitle,
  period,
  points,
}: {
  title: string;
  subtitle: string;
  period: string;
  points: string[];
}) {
  return (
    <div className="relative pl-6">
      <div className="absolute left-0 top-1 h-3 w-3 rounded-full bg-teal shadow-glow" />
      <h3 className="font-medium text-white/95">{title}</h3>
      <div className="text-white/70 text-sm">{subtitle} • {period}</div>
      <ul className="mt-2 list-disc space-y-1 pl-5 text-white/80 text-sm">
        {points.map((p, i) => <li key={i}>{p}</li>)}
      </ul>
    </div>
  );
}

8) Cómo usar la imagen de referencia

Guardá tu imagen en public/ref.jpg (o el nombre que prefieras).

Si querés mostrar un header hero con la referencia tenue como textura:

// ejemplo rápido dentro de <header>
<div
  className="relative -mx-6 mb-6 h-24 rounded-2xl overflow-hidden"
  style={{
    backgroundImage: "url(/ref.jpg)",
    backgroundSize: "cover",
    backgroundPosition: "center",
    filter: "grayscale(100%) opacity(0.15)",
  }}
/>


La idea es mantenerla muy sutil para no competir con el texto.

9) Micro-interacciones (si ya tenés framer-motion)

Podés envolver Card con un whileHover={{ y: -2 }} y transition={{ type: "spring", stiffness: 260, damping: 20 }} para un lift suave.

O añadir un focus ring con la utilidad ring-acc en hover.
